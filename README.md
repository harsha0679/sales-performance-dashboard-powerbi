# Sales Performance Dashboard using Power BI

## Project Overview

This project focuses on analyzing sales performance across different countries, products, and sales representatives using Power BI. The dashboard transforms raw sales data into actionable business insights, enabling stakeholders to monitor key performance indicators and make data-driven decisions.

---

## Business Objective

The objective of this project was to create an interactive dashboard that helps management:

- Monitor overall sales performance
- Track sales trends over time
- Identify top-performing products
- Analyze country-wise sales contribution
- Evaluate salesperson performance
- Support business decision-making through data visualization

---

## Tools & Technologies

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## Data Preparation

The dataset was cleaned and transformed using Power Query. The following steps were performed:

- Checked and handled missing values
- Verified data types
- Removed unnecessary columns
- Renamed columns for better readability
- Formatted date fields
- Validated sales and quantity data

---

## Key Performance Indicators (KPIs)

### Total Sales Amount
Measures the overall revenue generated from sales transactions.

### Total Boxes Sold
Tracks the total quantity of products sold.

---

## Dashboard Features

### Monthly Sales Trend Analysis
- Visualized sales performance over time using a line chart.
- Helped identify growth trends and sales fluctuations.

### Product Performance Analysis
- Compared revenue generated by different products.
- Identified top-selling and high-revenue products.

### Country-wise Sales Analysis
- Analyzed revenue contribution from different countries.
- Helped identify the strongest performing markets.

### Salesperson Performance Analysis
- Evaluated individual sales performance.
- Highlighted top-performing sales representatives.

### Interactive Date Filtering
- Enabled users to analyze data for custom date ranges.
- Improved dashboard interactivity and flexibility.

---

## Key Insights

- The UK contributed the highest share of total sales.
- The USA and India were among the top-performing markets.
- A small number of products generated a significant portion of total revenue.
- Sales performance varied across different months.
- Certain sales representatives consistently outperformed others.

---

## DAX Measures Used

```DAX
Total Amount = SUM(Sales[Amount])

Total Boxes = SUM(Sales[Boxes])
```
